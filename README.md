# DevOps Automation Scripts

Real-world DevOps automation scripts built during 30-day interview prep.

## Scripts
- `disk_monitor.sh` — Bash disk usage monitor with alerts
- `list_ec2.py` — List AWS EC2 instances using boto3
- `ec2_manager.py` — Full EC2 lifecycle automation (create/stop/terminate)
- `s3_manager.py` — S3 bucket operations (create/upload/download/delete)

## Tech Stack
Python 3.14 · boto3 · Bash · AWS CLI
